# Story 1.1: Setup e Estrutura Base do Site Institucional

**Epic:** 1 - Site Institucional
**ID:** STORY-1.1
**Status:** Done
**Data Criação:** 26/02/2026

---

## 📋 Descrição

Configurar a estrutura base e infraestrutura inicial para o desenvolvimento do site institucional, incluindo:
- Stack tecnológico definido e instalado
- Repositório e estrutura de pastas configurados
- Template HTML/CSS base responsive
- Ferramentas de build, linting e testes configuradas
- CI/CD pipeline básico funcional
- Ambiente de desenvolvimento local pronto

---

## ✅ Critérios de Aceitação

**Given** um computador desenvolvedor com Node.js 18+ instalado
**When** os comandos de setup são executados (`npm install`, `npm run dev`)
**Then**:
- [ ] Repositório Git inicializado com estrutura padrão
- [ ] Todas as dependências instaladas sem erros
- [ ] `npm run dev` inicia servidor local em http://localhost:5173
- [ ] `npm run build` gera build otimizado
- [ ] `npm run lint` executa sem erros críticos
- [ ] `npm run test` roda testes com cobertura >= 50%
- [ ] Template base responsivo carrega em < 1s (desktop)
- [ ] CSS framework escolhido e documentado
- [ ] Variáveis de ambiente configuradas (.env.example)
- [ ] README com instruções de setup completo

---

## 🎯 Escopo

### ✔️ Incluso (IN)

1. **Stack Tecnológico**
   - Frontend: React 18+ ou Next.js 14+
   - Estilos: Tailwind CSS ou Styled Components
   - Build: Vite ou Next.js built-in
   - Testes: Vitest + React Testing Library
   - Linting: ESLint + Prettier

2. **Estrutura de Pastas**
   ```
   .
   ├── src/
   │   ├── components/
   │   ├── pages/
   │   ├── styles/
   │   ├── utils/
   │   └── App.jsx
   ├── public/
   ├── tests/
   ├── .github/workflows/
   ├── .env.example
   ├── vite.config.js (ou next.config.js)
   ├── package.json
   └── README.md
   ```

3. **Componentes Base**
   - Layout principal (Header, Footer, Main)
   - Componente Hero para homepage
   - Componente Card reutilizável
   - Componente Form base

4. **CI/CD**
   - GitHub Actions workflow (lint, test, build)
   - Deploy automático para staging

### ❌ Não Incluso (OUT)

- Conteúdo real da empresa (textos, imagens)
- Funcionalidades de formulário (apenas estrutura)
- SEO avançado
- Integração com banco de dados
- Temas Dark mode (v1)
