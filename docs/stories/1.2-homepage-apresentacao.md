# Story 1.2: Homepage com Apresentação da Empresa

**Epic:** 1 - Site Institucional
**ID:** STORY-1.2
**Status:** Done
**Data Criação:** 26/02/2026

---

## 📋 Descrição

Desenvolver a página inicial (homepage) do site institucional com seções que apresentam a empresa, sua missão, valores e diferencias, incluindo CTA (Call-To-Action) estratégicos para conversão.

---

## ✅ Critérios de Aceitação

**Given** um usuário visitando o site pela primeira vez
**When** a homepage carrega
**Then**:
- [x] Hero section exibe com imagem/vídeo + headline principal e CTA
- [x] Seção "Sobre Nós" destaca missão, visão e valores
- [x] Seção "Por que nos escolher" lista 4-6 diferenciais com ícones
- [x] Seção "Últimos Projetos" exibe 3-4 case studies em cards
- [x] CTA secundário "Entre em Contato" aparece em 2+ seções
- [x] Footer contém: links rápidos, redes sociais, email, telefone
- [x] Design 100% responsivo (mobile, tablet, desktop)
- [ ] Performance: Lighthouse score >= 90
- [ ] SEO: Meta tags (title, description, og:*) configurados
- [ ] Tempo carregamento < 2.5s (3G throttling)

---

## 🎯 Escopo

### ✔️ Incluso (IN)

1. **Hero Section**
   - Headline + subheadline
   - CTA primário (ex: "Conheça Nossos Serviços")
   - Background image/vídeo otimizado

2. **Seção Sobre**
   - Texto descritivo
   - Missão, Visão, Valores (cards)
   - Números/métricas (projetos, clientes, anos)

3. **Seção Diferenciais**
   - 4-6 cards com ícone + título + descrição
   - Animações ao scroll (fade-in)

4. **Seção Projetos Preview**
   - Grid com 3-4 projects destacados
   - Link "Ver Portfólio Completo"

5. **CTA Section**
   - "Pronto para começar?" + botão contato

6. **Footer**
   - Logo + copyright
   - Links rápidos (Home, Portfólio, Blog, Contato)
   - Redes sociais (LinkedIn, Twitter, Instagram)
   - Email + Telefone

### ❌ Não Incluso (OUT)

- Blog ou artigos
- Funcionalidade real de contato (apenas form estrutura)
- Login/autenticação
- Conteúdo personalizado por usuário
