# Story 1.5: Sistema de Blog e Artigos

**Epic:** 1 - Site Institucional
**ID:** STORY-1.5
**Status:** Done
**Data Criação:** 26/02/2026

---

## 📋 Descrição

Desenvolver a seção de Blog do site para publicação de artigos, notícias e conteúdos relevantes (estabelecendo a empresa como autoridade no assunto). A página Listagem (com filtros) e a página do Artigo individual precisam ser criadas com boa usabilidade e forte apelo SEO.

---

## ✅ Critérios de Aceitação

**Given** um visitante interessado em conteúdo
**When** acessa a seção de Blog
**Then**:
- [x] A página principal lista os artigos mais recentes em cards
- [x] É possível filtrar artigos por Categoria (ex: Tecnologia, Design, Negócios)
- [x] Existe uma barra de busca funcionando
- [x] O artigo individual (`/blog/:slug`) renderiza título, data, autor, tempo de leitura e conteúdo rico
- [x] Artigos relacionados são exibidos no final de cada post
- [x] O layout é 100% responsivo
- [x] CTA (Call-to-Action) no meio ou fim do artigo convida para o serviço
- [x] SEO configurado corretamente (Meta Title, Description dinâmica por artigo)

---

## 🎯 Escopo

### ✔️ Incluso (IN)

1. **Página de Listagem (/blog)**
   - Hero/Header do Blog
   - Filtro de categorias
   - Grid/Lista de posts (com paginação/load more simulado)
   - Barra de pesquisa

2. **Página Interna do Artigo (/blog/:slug)**
   - Cabeçalho rico com Imagem de Capa
   - Corpo do texto suportando formatação HTML/Markdown simulada
   - Informações do Autor e Data
   - Componente "Compartilhar nas redes" (estático)
   - Bloco de "Artigos Relacionados"

3. **Gerenciamento de Dados (Frontend Fake-Backend)**
   - JSON com array de posts para simular o banco de dados
   - Utilização de React Router para passagem de parâmetros (`slug`)

### ❌ Não Incluso (OUT)

- Painel Administrativo de Criação de Posts (CMS real não no MVP)
- Sistema dinâmico de comentários
- Likes ou métricas sociais interativas em tempo real
- Feed RSS
